## Quick Start

1. Read the [setup guide](whatsapp-business-api-setup.md)
2. Configure your `PHONE_NUMBER_ID` and `ACCESS_TOKEN` in `send-test-message.sh`
3. Run: `./send-test-message.sh <recipient_phone_number>`
