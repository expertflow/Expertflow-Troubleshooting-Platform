CC User



Object Name

Agent (will be renamed to CCUser)

Description

This object contains agent information





Parameter

Description

id

This is the UUID in the format. 

participantType

This will always be CCUser

keycloakUser

User credentials of the associated KeyCloak user.

associatedRoutingAttributes

Routing attributes associated with this CCUser.



KeyClockUser



Id

This is the UUID in the format. 

FirstName

First name as defined in KeyCloak

LastName

Last name as defined in KeyCloak

roles

Array of roles assigned to this user in KeyCloak

username

As defined in KeyCloak

permittedResources

An array of permitted resources along with scopes as defined in KeyCloak

realm

The name of the company or the organization unit. (default-tenant)
