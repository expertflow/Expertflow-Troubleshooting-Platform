Release Notes

This release is mainly focused on refining the Caller List component, addressing UX issues, and conducting comprehensive load tests for Easy Announcement, Business Calendar, and Caller List.

New in Release 13.5

Features

Description

Caller List Component

Optimized Caller Data Upload: Implemented optimizations to reduce the time taken for uploading caller data, resulting in improved efficiency and faster data processing.

Database TTL Data Removal: Fixed the issue where numbers were not being removed from the database after passing the TTL (Time-To-Live) date. Adjustments were made to ensure proper data deletion based on expiration criteria.

Expanded Digit Limits for Regions: Addressed limitations on the number of digits allowed for regions. This enhancement enables the system to accommodate longer region codes up to 13 digits with and without + sign.

Load test

A comprehensive load test has been run for the Caller List, Easy Announcement and Business Calendar. Detailed reports can be viewed here.
