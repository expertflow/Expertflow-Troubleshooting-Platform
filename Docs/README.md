## 📝 Purpose

This folder is a **flexible catch-all** for documentation that doesn't fit neatly into the other top-level modules:

- API reference guides
- Operational runbooks
- Onboarding checklists
- Architecture decision records (ADRs)
- Meeting notes & retrospectives

> 💡 **Tip:** As this repository grows, create subfolders here to organize content by domain (e.g., `Docs/API/`, `Docs/Runbooks/`, `Docs/ADRs/`).
