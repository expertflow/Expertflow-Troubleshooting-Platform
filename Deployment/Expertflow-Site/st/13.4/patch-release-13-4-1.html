Patch Release (13.4.1)

This release covers compatibility issues with UCCE 12.6, bug fixes, and some UX issues.

Description

Status

QA-PASSED

Release Summary

Resolved some issues in the 13.4.1 release

Upgrade Guide

Upgrade Guide 13.4 to 13.4.1

The following features have been introduced in this release.

Features

Description

SMBv2 Protocol Support

The Prompt Component now supports the SMBv2 protocol, enhancing security and reliability in prompt access. It is compatible with UCCE 12.6.

High Availability

Supervisor tool functionality has been thoroughly tested in High Availability (HA) configurations with UCCX 12.0.

Single Sign-On (SSO)

Single Sign-On (SSO) functionality has been fixed for finesse users, allowing them to seamlessly log in to the Supervisor tool (UCCX 12.0).
